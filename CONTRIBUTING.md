# 🌍 Contributing to Bug-Free Travel Guides

We welcome all contributions! Here is how to get involved:

## Add a New Destination

1. **Open a GitHub Issue** with title: `New Destination: [Place Name]`
2. Include:
   - Country/Region
   - Your bug experience there
   - Best/worst seasons for insects (with months)
   - Why it is bug-free (climate, altitude, wind, etc.)
   - Any rare exceptions
   - One travel tip for fellow bug-avoiders
3. A maintainer will review and format it into a proper guide!

## Improve Existing Guides

- Found outdated info? Open a PR with corrections
- Have a better travel tip for a featured destination? Share it!

## Community Values

- Be kind — bug phobias are valid
- Be accurate — cite your sources
- Be helpful — actionable tips only
- No horror stories — we are solution-oriented
